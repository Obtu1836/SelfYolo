# SelfYolo — v1 说明

🔎 项目定位

- SelfYolo v1 是一个模块化、轻量的 YOLO-like 单网格目标检测实现，代码入口与核心文件在本目录内。主要用于学习yolov1的思想

🧩 结构概览

- backbone / neck / head / matcher / loss 清晰解耦，配置驱动，便于试验和扩展。

✨ 与原版（经典 YOLO）主要不同

- ⚙️ 骨干替换：使用 `Resnet` 作为特征提取骨干，提升特征表达与训练稳定性。 
- 🧠 Neck 强化：采用 `SPPF` 结构以扩大感受野并提升上下文融合能力。 
- 🔀 Head 分支化：分类与回归各用独立子网，减少相互干扰并便于调参。 
- 🎯 匹配策略：按目标中心映射到特征图格子（更接近 YOLOv1 的思路），不使用 anchor 匹配复杂逻辑。 
- 📈 回归鲁棒性：使用 GIoU 提升边框回归效果。
